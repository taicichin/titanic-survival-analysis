# titanic-survival-analysis
Exploratory data analysis of the Titanic dataset

Hi, I'm a data science learner, and this is my first small project, guided by Claude as a study assistant. This project uses a dataset from the Titanic disaster, and its main goals are to analyse whether age, class, and sex influence the survival rate. After running the analysis, it was possible to draw the following conclusions:

1. The average rate of survival was 38%, while for women it was almost double (74%)
2. The rate of survival was directly related to age, where for children it was 58%, decreasing to around 40% in adult ages and plummeting to 22% at senior age.
3. The same pattern was observed for the class passengers were traveling in, with higher rates for first class and lower for third class.
